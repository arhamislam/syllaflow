# Syllaflow
> Syllaflow is a Chrome extension that uses AI to extract deadlines from syllabus PDFs and exports them directly into Notion workspaces.

## Tech Stack
- **Languages:** JavaScript, HTML/CSS
- **Frameworks:** Node.js, Firebase, Bootstrap
- **Build:** Vite, @crxjs/vite-plugin
- **Libraries:** PDF.js
- **APIs:** Gemini AI, Notion API

## Features
- **AI-Powered Extraction:** Uses Gemini AI to intelligently identify assignment names, due dates, and weighting from PDF text.
- **Notion Integration:** Automatically integrates to your Notion Workspace and populates your Notion Database.
- **Secure Login:** User authentication and preferences managed via Firebase.

## Prerequisites
Before running this project locally, ensure you have the following installed:
- Node.js (v24 or higher)
- npm (v11 or higher)

You will also need API keys for:
- Gemini AI
- Firebase Project
- Notion Integration

## Development Setup
1. Clone the repository:
    ```
    git clone https://github.com/arhamislam/syllaflow.git
    cd syllaflow
    ```

2. Install the dependencies: `npm install`

3. Configure the environment by creating a `.env` file.

4. Build the extension: `npm run build`

5. Load into Chrome.
    1. Go to `chrome://extensions/`.
    2. Toggle Developer Mode.
    3. Click Load Unpacked.
    4. Select the `dist` folder generated by the build command.

## Usage
1. Download a syllabus as a PDF.
2. Click the Syllaflow extension icon.
3. Upload your syllabus PDF.
4. Select your Notion Workspace, then click "Export to Notion".
